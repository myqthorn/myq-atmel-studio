void nRF24L01::enableACKPayload ( )

Enables payload with ACK.

If ACK packet payload is activated, ACK packets have dynamic payload lengths, and the Dynamic Payload Length feature should be enabled for pipe 0 on the PTX and PRX. This is to ensure that they receive the ACK packets with payloads. If the payload in ACK is more than 15 byte in 2Mbps mode, the ARD must be 500μS or more. If the payload is more than 5 byte in 1Mbps mode the ARD must be 500μS or more.
